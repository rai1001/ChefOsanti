# QA Visual Checklist

## Tokens
- [ ] No hay colores hex hardcodeados (salvo en theme/tokens)
- [ ] Borde estándar aplicado en cards/tables
- [ ] Shadows coherentes
- [ ] Radius coherente (2xl)

## Tipografía
- [ ] H1 consistente
- [ ] Secondary text consistente
- [ ] Tablas: tamaño mínimo legible

## Estados
- [ ] Hover en filas/celdas
- [ ] Focus visible en inputs/buttons
- [ ] Loading skeletons
- [ ] Empty states con mensaje y CTA

## Kitchen Mode
- [ ] Targets 44px+
- [ ] Contraste aumentado
- [ ] Texto más brillante

## Multi-branch
- [ ] Sucursal activa visible
- [ ] Acciones destructivas muestran branch + item
